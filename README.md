# Login Authentication System

This is a simple login authentication project created using HTML, CSS and JavaScript.

## About the Project
In this project, I have created a basic login page where a user can enter a username and password. The system checks whether the entered credentials are correct and displays a success or error message.

## Features
- Simple login form
- Username and password validation
- Success and error messages

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Files
- login.html → Structure of the login page
- login.css → Styling of the page
- login.js → Login validation logic

## How to Run
1. Download all the files
2. Open login.html in any web browser
3. Enter the login details and check the result
